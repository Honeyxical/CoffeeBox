#  CoffeeBox

Project objective: Realized the software for making coffee and implementing the payment of the LEI700 coffee machine.

Realized:
* Display user menu
* Display service menu
* Make Espresso
* Make Americano
* Make Cappuccino
* Balance operation
* Income operation
* Counting cups
* Block system 
* PIN authentication

Technology stack: C++
